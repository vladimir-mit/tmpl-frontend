# tmpl-frontend

Заготовка для верстки. Mobile First.

Используется Gulp и SCSS.
Можно добавить Jquery, bxSlider, Fancybox, Bootstrap 4, Balloon-scss, normalize.css, magnificPopup, Slick-carousel, overlayScrollbars.

- gulp-concat
- gulp-autoprefixer
- gulp-cssnano
- gulp-uglify
- gulp-sass
- gulp-watch
- gulp-sourcemaps
- gulp-clean
- gulp-rigger
- browser-sync
